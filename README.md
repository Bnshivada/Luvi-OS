## 🚀 Proje Hakkında
**LuviOS**, Windows 10'un hantal yapısını cerrahi bir operasyonla temizleyen, Zorin OS'in modern estetiğini Windows dünyasına taşıyan açık kaynaklı bir modifikasyon projesidir. Özellikle düşük depolama alanı ve yüksek performans hedefleyen oyuncular için geliştirilmiştir.

### 🎯 Neden LuviOS?
Birçok custom ISO projesinin aksine LuviOS, kullanıcıların karşılaştığı donanımsal kararsızlıklara (özellikle **VIDEO_TDR_FAILURE**) karşı özel optimizasyonlar içerir.

---

## ✨ Öne Çıkan Özellikler

- 📉 **Ultra-Low Storage:** Sistem dosyaları `compact.exe` ile sıkıştırılmış, gereksiz servisler (bloatware) tamamen kaldırılmıştır.
- 🏎️ **Gaming Engine:** GPU önceliği ve işlemci zamanlaması oyunlar için optimize edildi.
- 🛡️ **TDR Stability:** Ekran kartı sürücü çökmelerini engellemek için özelleştirilmiş `TdrDelay` yapılandırması.
- 🎨 **Aesthetic:** Modern, minimalist ve göz yormayan karanlık tema arayüzü.
- 🚫 **No Spyware:** Telemetri ve arka plan takip servisleri devre dışı bırakıldı.

## 🛠️ Klasör Yapısı

| Klasör | İçerik |
| :--- | :--- |
| `Core/` | Windows servislerini ve bileşenlerini budayan ana scriptler. |
| `Fixes/` | Mavi ekran (BSOD) ve sürücü stabilite yamaları. |
| `UI/` | Zorin OS stili ikonlar, temalar ve kayıt defteri ayarları. |
| `Tools/` | ISO derleme sürecinde yardımcı olacak araçlar. |

## 🔧 Kurulum ve Kullanım

> [!IMPORTANT]
> LuviOS şu an geliştirme aşamasındadır. Scriptleri mevcut sisteminizde çalıştırmadan önce mutlaka yedek alın!

1. Bu depoyu klonlayın: `git clone https://github.com/Bnshivada/Luvi-OS.git`
2. `Core/` dizinindeki optimizasyon scriptlerini yönetici olarak çalıştırın.
3. Görünüm ayarları için `UI/` klasöründeki yönergeleri takip edin.
